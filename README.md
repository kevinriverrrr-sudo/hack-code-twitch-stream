# Взломай Код - Twitch Stream Game

Интерактивная игра в прямом эфире на Twitch. Зрители угадывают 4-значный код в чате.

## Как работает

1. **Нейросеть** генерирует пароль (4 цифры) и подсказку
2. **Баллы начинаются со 100,000** и понижаются со временем (500/сек)
3. **5 исключённых цифр** показываются на экране (их точно нет в пароле)
4. Зрители пишут 4-значные числа в **чат Twitch**
5. Первый угадавший получает текущие баллы
6. Победитель может написать сообщение на экране

## Установка и запуск

```bash
npm install
node all-in-one.js
```

Или через detached launcher:
```bash
node launcher.js
```

## Настройка

Все настройки в файле `.env`:
- `TWITCH_STREAM_KEY` - ключ стрима Twitch
- `TWITCH_CHANNEL` - название канала
- `FW_KEY` - API ключ Fireworks AI
- `BASE_POINTS` - начальное количество баллов (по умолчанию 100000)

## Стек технологий

- **Node.js** + Canvas (рендеринг графики)
- **FFmpeg** (RTMP стриминг на Twitch)
- **tmi.js** (Twitch чат интеграция)
- **better-sqlite3** (база данных игроков и раундов)
- **Fireworks AI** (генерация паролей)

## Важно: Сохранение трансляций на Twitch

Чтобы трансляции сохранялись как VOD:
1. Зайди на https://www.twitch.tv/settings/stream
2. Включи **"Store past broadcasts"** (Сохранять прошлые трансляции)
3. Трансляции будут храниться на Twitch 14-60 дней

## Файловая структура

- `all-in-one.js` - Основной код игры (Canvas + FFmpeg + Twitch + DB + AI)
- `launcher.js` - Detached launcher для фонового запуска
- `public/` - Статические файлы (админка, фронтенд)
- `.env` - Настройки (ключи, параметры)
