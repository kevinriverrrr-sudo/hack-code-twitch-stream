# 🔐 Взломай Код - Twitch Stream Game

Интерактивная игра в прямом эфире на Twitch. Зрители угадывают 4-значный код в чате, пока баллы понижаются!

## 🎮 Как работает

1. **Нейросеть (AI)** генерирует пароль (4 цифры) и подсказку каждый раунд
2. **Баллы начинаются со 100,000** и понижаются со временем (500/сек)
3. **5 исключённых цифр** показываются на экране (их точно нет в пароле)
4. Зрители пишут 4-значные числа в **чат Twitch**
5. Первый угадавший получает текущие баллы
6. Победитель может написать сообщение на экране
7. Если победитель воздерживается — отображается соответствующее сообщение

## ⚡ Быстрый старт

```bash
# Установить зависимости
npm install

# Запустить (фоновый режим через launcher)
node launcher.js

# Или запустить напрямую
node all-in-one.js
```

## ⚙️ Настройка

Все настройки в файле `.env`:

| Переменная | Описание | По умолчанию |
|---|---|---|
| `TWITCH_STREAM_KEY` | Ключ стрима Twitch | - |
| `TWITCH_CHANNEL` | Название канала | `mazafakezo` |
| `TWITCH_USER_ID` | ID пользователя Twitch | - |
| `TWITCH_AUTH_TOKEN` | Auth token для Twitch API | - |
| `STREAM_TITLE` | Название трансляции | - |
| `STREAM_CATEGORY_ID` | ID категории (509670 = Just Chatting) | `509670` |
| `FW_KEY` | API ключ Fireworks AI | - |
| `BASE_POINTS` | Начальное количество баллов | `100000` |
| `POINTS_DECREASE_RATE` | Понижение баллов в секунду | `500` |
| `MIN_POINTS` | Минимальные баллы за раунд | `1000` |
| `FPS` | FPS стрима | `10` |

## 🔧 Обязательно: Включить сохранение трансляций (VOD)

Чтобы трансляции сохранялись на Twitch и их можно было пересматривать:

1. Зайди на **https://www.twitch.tv/settings/stream**
2. Включи переключатель **"Store past broadcasts"** (Сохранять прошлые трансляции)
3. Трансляции будут храниться **14-60 дней** на Twitch

⚠️ Это нужно сделать вручную, так как API не поддерживает эту настройку программно.

## 🔧 Установить название и категорию стрима

2. Название: `Взломай Код Если Сможешь | Интерактивная Игра в Прямом Эфире`
3. Категория: `Just Chatting`

## 📁 Файловая структура

```
├── all-in-one.js      # Основной код (всё в одном файле)
├── launcher.js         # Detached launcher для фонового запуска
├── .env               # Настройки (ключи, параметры)
├── package.json        # Зависимости Node.js
├── public/             # Статические файлы
│   ├── admin.html      # Админ-панель
│   ├── index.html      # Главная страница
│   └── logo.svg        # Логотип
└── game.db             # SQLite база данных (создаётся автоматически)
```

## 🛠 Стек технологий

| Компонент | Технология | Назначение |
|---|---|---|
| Рендеринг | Node.js Canvas | Отрисовка графики 1280x720 |
| Стриминг | FFmpeg + RTMP | Трансляция на Twitch |
| Чат | tmi.js | Чтение сообщений Twitch |
| База данных | better-sqlite3 | Хранение игроков и раундов |
| AI | Fireworks AI | Генерация паролей и подсказок |

## 🎨 Дизайн

Киберпанк-стилистика с:
- Matrix-дождь из цифр на фоне
- Неоновые свечения (голубой/зелёный/красный)
- Сканлайны и глитч-эффекты
- Анимация пульса на слотах пароля
- Красная вспышка + тряска при неправильном ответе
- Зелёная вспышка + фейерверк при взломе
- Анимированный лидерборд с медалями
- Twitch чат с цветными никами

## 📊 База данных

Автоматически создаётся `game.db` (SQLite) с таблицами:
- **players** — игроки: ник, аватар, очки, количество взломов
- **rounds** — раунды: пароль, подсказка, победитель, время, очки
